## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/portfolio.git
```

Move into project directory:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
